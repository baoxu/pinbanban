## 三、交互说明

### 3.1 核心交互流程

1. **参数选择**：滑块 + 实时预览，AI 推荐高亮显示
2. **图案选择**：网格展示，点击预览大图，支持 360° 旋转查看
3. **拼单进度**：实时更新，成团时推送通知 + 动画庆祝
4. **分享邀请**：生成海报 + 小程序码，好友扫码直接加入拼单

### 3.2 状态反馈

| 状态 | 展示 | 用户操作 |
|------|------|---------|
| 拼单中 | 进度条 + 倒计时 | 分享邀请、等待 |
| 拼单成功 | 庆祝动画 + 生产时间线 | 支付尾款（如付定金） |
| 拼单失败 | 退款通知 + 推荐相似拼单 | 加入其他拼单或退款 |
| 生产中 | 工厂直播/照片更新 | 查看进度 |
| 已发货 | 物流追踪 | 确认收货 |

## 四、视觉规范

### 4.1 配色方案

```
主色：    #1E90FF  (冰雪蓝)
辅助色：  #FF6B6B  (活力红)
背景色：  #F8F9FA  (雪白色)
文字色：  #2D3436  (深灰)
强调色：  #FFE66D  (阳光黄)
```

### 4.2 字体规范

- 标题：PingFang SC Bold / 24px
- 副标题：PingFang SC Medium / 18px
- 正文：PingFang SC Regular / 14px
- 辅助文字：PingFang SC Light / 12px

### 4.3 图标风格

- 线性图标，2px 描边
- 圆角设计，亲和力强
- 雪板、雪山、雪花等冰雪元素
