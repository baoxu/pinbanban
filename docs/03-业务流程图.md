## 四、关键节点说明

### 4.1 拼单成功判定

| 条件 | 说明 |
|------|------|
| 人数达标 | 达到设定拼单人数（2-5 人） |
| 时间未超时 | 在设定时间内（默认 72 小时） |
| 定金支付 | 所有成员支付定金 |

### 4.2 拼单失败处理

| 场景 | 处理方式 |
|------|---------|
| 超时未成团 | 自动退款 + 优惠券补偿 |
| 用户主动退出 | 扣除定金（作为违约金） |
| 工厂无法生产 | 全额退款 + 优先推荐其他板型 |

### 4.3 异常处理

| 异常 | 处理流程 |
|------|---------|
| 支付失败 | 重试 3 次 → 推送通知 → 订单保留 24 小时 |
| 物流延误 | 主动通知 + 补偿优惠券 |
| 质量问题 | 免费重做 + 承担往返运费 |
| 用户拒收 | 协商退货 → 扣除运费 → 二次销售 |
